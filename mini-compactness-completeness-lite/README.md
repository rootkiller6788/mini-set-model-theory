# mini-compactness-completeness-lite

Compactness, Completeness, Löwenheim-Skolem, Shelah Main Gap, and the classification program.

## Structure

- `Core/` — Basic definitions, objects, and laws
- `Theorems/` — Compactness, Completeness, LS, Classification, Main Gap, Universal Properties
- `Examples/` — Standard examples and counterexamples
- `Bridges/` — Connections to topology, computation, algebra, and geometry
- `Morphisms/` — Elementary embeddings, isomorphisms, homomorphisms
- `Constructions/` — Ultraproducts, elementary extensions, diagrams, saturation
- `Properties/` — Stability, categoricity, omitting types

## Dependencies

- mini-function-relation
- mini-cardinal-ordinal
- mini-satisfaction-model
- mini-logic-kernel
