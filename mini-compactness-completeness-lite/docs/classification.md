# Classification Program

## Shelah's Classification Theory

The classification program, initiated by Saharon Shelah, aims to classify first-order theories by their model-theoretic complexity.

### Stability Hierarchy

Theories are classified along the stability spectrum:
- **Stable theories**: Well-behaved, few types.
- **Superstable theories**: Stronger stability condition.
- **ω-stable theories**: Morley's categoricity context.

### Main Gap Theorem

Shelah proved that the spectrum function I(T, κ) — the number of non-isomorphic models of T of cardinality κ — either grows maximally (2^κ) or is bounded by ℶω₁ for all uncountable κ.

### Key Results

- Classifiable theories have structure theorems (prime models over independent trees).
- Non-classifiable theories have maximal spectrum.
- The dividing line: classifiability is equivalent to having a structure theorem.
