# MiniZFCLite Documentation

Welcome to the `mini-zfc-lite` package documentation.

## Overview

`mini-zfc-lite` provides the 8 Zermelo-Fraenkel axioms with Choice (ZFC)
expressed as `PredFormula` values and organized into an `AxiomSystem`.
